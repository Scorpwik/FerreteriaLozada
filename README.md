# Web Ferretería Lozada 🛠️

Proyecto final de Desarrollo Web - USFQ.
**Integrantes:** Juan Diego Cadena, Joshua Lozada, Raymond Portilla.

## Objetivo
Crear una plataforma con catálogo interactivo y carrito de cotización vinculado a WhatsApp para microempresas y maestros de obra.

## Identidad Visual
- **Colores:** Naranja, Negro, Blanco.
- **Tipografías:** Zen Dots (Primaria), Anta (Secundaria).
- **Tono:** Amigable, experto y "de vecino".

## Entregables Finales
- Código fuente (HTML/CSS/JS).
- Diseño responsivo (Móvil/Desktop).
- Carrito con API de WhatsApp.
